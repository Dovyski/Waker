Columns and images

Waker has a set of pre-configured css classes that can be used to organize texts, images and columns. You can create a column using the following structure:

		
  <div class="large-column float-left">
      <p> (...) </p>
  </div>

There are four classes of columns: full-column (fill the entire page width), large-column, medium-column and small-column. Excluding the full-column all other columns can float left or right in the page since they don't fill it completly. In order to make a column float left or right, add the float-left or float-right classes.

Images can be positioned using the media-right, media-left, media-center and media-headline classes. Those classes also resize the image to make it fit better into the page. If an image has no class, then it will be displayed as-is.

		
  <img src="path/img.png" class="media-right" />

The following sections demonstrate the positioning classes.

Image demonstration

The image at the beginning of this dossier (the big orange one) has the media-headline class in action. It makes the image fill completly the page, adjusting its size to fit the area.

The image of an ant featured above has the media-right class in action, so it will float to the right of the page. Its size will be adjusted to fit the page. The image on the left of this text has the media-left class.

The image below has the media-center class in action, which will make the image to adjust its size to fill almost all page content.
