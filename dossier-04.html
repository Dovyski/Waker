Enhanced content

Waker has a set of custom tags that helps you create enhanced content such as styled blockquotes or embed QR codes. All custom tags are prefixed with w:, such as <w:end />.

Below is a list and description of each Waker custom tag.

Quotes

If your text has a quote, you can use the <w:quote /> tag to display it. All text within that custom tags will be displayed between quote marks, inside a box.

The code below will produce the quotes on the right:

Look mom, some text surrounded by nice quote marks. Cool! You can use can place your quote tags inside any column size. 

  <w:quote> (text) </w:quote>

QR Code

Sometimes is useful to place a QR code in the text to help readers rich an URL using their mobile devices. If you want convert an URL into a QR code image, use the <w:qr /> custom tag.

The following code...

		
  <w:qr>http://dovyski.github.com/Waker</w:qr>

...will produce this image:

http://dovyski.github.com/Waker

You can adjust the width and height of the QR image using the width and height atrributes, the same way you use them to adjust the size of an <img /> tag. E.g.:

		
  <w:qr width="300" height="300"> (...) </w:qr>

End mark

Several magazines use a special mark at the end of an article to indicate the end of that content. It is useful when an article uses several pages of the magazine and the author must instruct the reader to keep reading.

Waker has the <w:end /> custom tag which is replaced by a special sign that indicates the end of the current article.

The following code show how to use the custom tag:

		
  <p> This is the last paragraph of an article. <w:end/> </p>

It will produce the following result:

This is the last paragraph of an article.
