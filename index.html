Waker

It's time to start building HTML5 magazines!

About Some background history and information

Waker is a "web-only" version of Laker, a compendium of files, frameworks, styles and tips for designing digital publications. It is fully based on HTML5 and was designed to be easy and simple to use.

Why?

Because I thought Laker and Baker could not (easily) do what I wanted to do: an eletronic magazine, available in a variety of forms.

Features

Minimum configuration; uses HTML5, JS and CSS only; magazine-like layout; (almost) automatic index generation; no JS skills required (only HTML/CSS).

Quick-start Waker crash course

Create an online publication (e.g. magazine) using Waker in 4 steps:

  1. Download Waker.
  2. Add a content page (e.g. dossier01.html).
  3. Create a link for that page in config.html.
  4. You are done, take a coffee!
